# The Ultimate Gym & Health Club

A premium, fully responsive gym website built using HTML, CSS and JavaScript.

## Features

- Premium Dark Luxury UI
- 100% Responsive
- Glassmorphism Design
- Membership Plans
- Google Maps
- WhatsApp One-Click Contact
- Call One-Click Contact
- Instagram & Facebook Links
- BMI Calculator
- Water Intake Calculator
- Calorie Calculator
- Lung Challenge
- Daily Motivation
- Testimonials
- Gallery
- FAQ
- SEO Ready
- GitHub Ready
- Cloudflare Pages Ready


## Customization

To use this website for another gym:

- Replace the logo inside `images/`
- Replace gym images inside `images/`
- Update the gym name
- Update the phone number
- Update the WhatsApp number
- Update the address
- Update the membership plans
- Update Instagram and Facebook links

The website is then ready to publish.

## Deployment

1. Upload to GitHub.
2. Connect the repository to Cloudflare Pages.
3. Publish.
4. Add your custom domain (optional).
